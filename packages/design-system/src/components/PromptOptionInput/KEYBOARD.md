# Picker — keyboard guide

Pick one or more options to continue.

- `↑` `↓` to move, `Enter` to pick the highlighted option
- Search field at the top? Just type to filter the list
- `⌘ Enter` (or `Ctrl Enter`) submits — next step, or finish on the last one
- `Alt ←` `Alt →` jump between steps when there's more than one
- `Esc` or click × to close and return to the prompt input
- Text box at the end? Type a custom value and press `Enter`

| Key | What it does |
| --- | --- |
| Type any letters | Filter the list |
| `↑` / `↓` | Move the highlight |
| `Enter` | Pick the highlighted option |
| `⌘ Enter` / `Ctrl Enter` | Submit and go to the next step (or finish on the last step) |
| `Alt ←` / `Alt →` | Go to the previous / next step (when available) |
| `Esc` | Clear what you typed, then close on the next press |

If you see a text box at the bottom, type your own value and press `Enter` to submit it.

To leave the picker and go back to the normal prompt input, press `Esc` or click the × in the top right.
